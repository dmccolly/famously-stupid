# Famously Stupid

The official propaganda outlet for the Ministry of Information.

## About
This is a satirical news and video platform, featuring:
- **Propaganda Reels:** 1940s-style newsreels with a modern, sarcastic twist.
- **Ballot Purification:** Our latest campaign for democracy.
- **The Register:** Updates from the front lines of famous stupidity.

## Development
- Hosted on Digital Ocean App Platform (static site).
- Uses simple HTML/CSS for now.
- Content managed via GitHub.
